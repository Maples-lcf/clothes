## 一、开始使用

### 1.1 打开系统

1. 在电脑浏览器（推荐 Chrome）地址栏输入系统地址
2. 本地开发环境一般为：`http://localhost:5173`
3. 打开后会看到登录页

### 1.2 登录

1. 输入**邮箱**和**密码**
2. 点击「登录」
3. 登录成功后自动进入「数据概览」首页

> 首次使用需注册账号，点击「立即注册」填写邮箱和密码即可。

### 1.3 认识界面

登录后你会看到：

- **左边深色菜单**：切换不同功能
- **上面白色顶栏**：当前页面名称、刷新、退出
- **中间白色区域**：具体操作内容

**常用菜单**：

| 菜单 | 什么时候用 |
|------|------------|
| 数据概览 | 看整体卖了多少、赚多少 |
| 商品管理 | 管理所有衣服 |
| 闲鱼导入 | 从闲鱼快速添加商品 |
| 销售记录 | 卖出后记账 |
| 闲鱼账号 | 管理多个闲鱼号 |
| 账号分析 | 看哪个号卖得好 |

## 二、第一次使用：基础设置

建议按以下顺序完成一次性配置（约 10 分钟）。

### 步骤 1：添加闲鱼账号

> 如果只有一个闲鱼号，也要先添加，后面记账才能选到。

1. 点击左侧 **「闲鱼账号」**
2. 点击右上角 **「新增账号」**
3. 填写：
   - **账号名称**：给自己看的名字，如「闲鱼主号」「清仓号」
   - **平台**：选「闲鱼」
   - **状态**：保持「启用」
   - **备注**：可选，如「卖新款」「清库存」
4. 点击 **「保存」**
5. 有几个号就重复添加几次

### 步骤 2：确认分类

1. 点击 **「分类管理」**
2. 系统默认已有：上衣、裤子、裙子、外套、配饰
3. 如需新增：点 **「新增分类」**，填名称和排序

### 步骤 3：录入现有商品

有两种方式（二选一或混用）：

- **方式 A**：从闲鱼粘贴链接导入 → 见 [第三章](#三从闲鱼导入商品)
- **方式 B**：手动新增 → 见 [第四章](#四手动管理商品)

## 三、从闲鱼导入商品

适合：已经在闲鱼上架了，想快速录入系统。

### 操作步骤

1. 在闲鱼 App 里，对商品点 **「分享」**，复制分享文案
2. 打开系统，进入 **「闲鱼导入」**
3. 把复制的文案 **粘贴** 到大文本框
4. 点击 **「获取商品信息」**
5. 等待几秒，下方会出现解析结果
6. **检查并修改**：
   - 商品名称、售价是否正确
   - **商品来源**：自己以前买的选「自有闲置」；进货的选「进货」并填进价
   - **闲鱼账号**：选这个商品在哪个号上卖
   - **分类**：选对应分类
   - **商品状态**：还在卖选「在售」
7. 点击 **「保存到商品管理」**

### 特殊情况：已经卖掉的商品

如果这件衣服**已经卖出**，导入时：

1. 商品状态选 **「售出下架」**
2. 填写 **运费**（你实际出的邮费）
3. 保存后系统会**自动记一笔销售**

## 四、手动管理商品

适合：新进货、还没上闲鱼、或需要精细编辑的商品。

### 4.1 新增商品

1. 进入 **「商品管理」**
2. 点击 **「新增商品」**
3. 填写基本信息：

| 字段 | 怎么填 |
|------|--------|
| 商品名称 | 必填，如「春季碎花连衣裙」 |
| 分类 | 选分类 |
| 商品来源 | **自有闲置**（自己以前买的）或 **进货**（有进价） |
| 闲鱼账号 | 选主要在哪个号卖 |
| 闲鱼链接 | 可选，有就粘贴 |
| 状态 | 一般选「在售」 |
| 备注 | 可选，写卖点等 |
| 商品图片 | 点「上传图片」 |

4. 添加 **SKU**（具体规格）：

| 字段 | 说明 |
|------|------|
| 颜色 | 如「白色」「均色」 |
| 尺码 | 如「M」「均码」 |
| 进价 | 自有闲置自动为 ¥0；进货需填写 |
| 售价 | 打算卖多少钱 |
| 库存 | 现在有几件 |
| 预警值 | 库存低于这个数会提醒，默认 2 |

5. 如果同一款有多个颜色/尺码，点 **「+ 添加 SKU」**
6. 点击 **「保存」**

### 4.2 编辑 / 删除商品

1. 在商品列表找到目标商品
2. 点 **「编辑」** 修改，或 **「删除」** 移除
3. 删除前系统会二次确认

### 4.3 搜索和筛选

列表上方可以按：

- 名称 / 货号
- 分类、状态、来源、闲鱼账号

快速找到商品。

### 4.4 批量导入商品

1. 点 **「模版下载」** 获取 Excel 模板
2. 按模板填写商品信息
3. 点 **「数据导入」** 上传文件

## 五、卖出后记账（最重要）

> **每卖出一单，记得回来记一笔。** 这样数据概览和账号分析才准确。

### 5.1 手动录入（推荐）

1. 进入 **「销售记录」**
2. 点击 **「录入销售」**
3. 点 **「选择商品」**，在弹窗里找到卖出的那款（可搜索）
4. 填写：

| 字段 | 怎么填 |
|------|--------|
| 销售渠道 | 闲鱼成交选「闲鱼」 |
| 成交账号 | **必选**，选是哪个闲鱼号卖出的 |
| 下单日期 | 默认今天，可改 |
| 订单状态 | 正常成交选「成功」；退款了选「有退款」 |
| 数量 | 一般填 1 |
| 成交额 | 实际收到的钱（买家付给你的） |
| 单件进价 | 自有闲置自动 ¥0；进货款会自动带出 |
| 平台服务费 | 闲鱼默认按 0.6% 估算，可手动改 |
| 运费 | 你出的邮费 |
| 其他费用 | 如有包装费等 |
| 备注 | 可选，如买家昵称 |

5. 看 **「预计利润」** 是否正确
6. 点击 **「保存」**

**保存后系统会自动：**
- 扣减对应 SKU 库存
- 计入该闲鱼号的销售统计
- 更新首页利润数据

### 5.2 批量导入销售

1. 点 **「模版下载」**
2. 按模板填写多笔销售
3. 点 **「数据导入」** 上传

### 5.3 修改已有记录

1. 在销售列表找到该笔记录
2. 点 **「编辑」**
3. 可修改：日期、账号、订单状态、实际售价、运费、平台费
4. 点 **「查看」** 可看完整详情

### 5.4 退款怎么处理

1. 找到该笔销售，点 **「编辑」**
2. 订单状态改为 **「有退款」**
3. 保存后，该笔**利润自动变为 ¥0**

> 注意：删除销售记录**不会**自动恢复库存，如有需要请去「库存管理」手动调整。

## 六、库存管理

### 6.1 查看库存

1. 进入 **「库存管理」**
2. 上方列表显示每个 SKU 的库存、进价、货值
3. 筛选 **「低库存」** 可快速看快卖完的款

### 6.2 调整库存

进货补货、盘点纠错时使用：

1. 找到对应 SKU，点 **「调整」**
2. 选择类型：
   - **入库**：新货到了，加库存
   - **出库**：损耗、赠送等，减库存
   - **调整**：直接改到正确数量
3. 填数量和备注
4. 点 **「确认」**

### 6.3 变动记录

页面下方可查看每次库存变化的记录，方便追溯。

## 七、闲鱼账号与流量投入

### 7.1 管理账号

在 **「闲鱼账号」** 页面可以：

- 新增 / 编辑 / 删除账号
- 停用暂时不用的号

### 7.2 记录流量投入

在闲鱼买了曝光、擦亮等推广时：

1. 在账号列表，点该账号的 **「流量投入」**
2. 点 **「新增投入」**
3. 填写日期、金额、备注（如「曝光推广」）
4. 保存

这些投入会在 **账号分析** 中参与净收益计算。

## 八、看数据和复盘

### 8.1 数据概览（首页）

打开系统第一眼就能看到：

| 卡片 | 含义 |
|------|------|
| 商品数 | 有多少在售商品 |
| SKU / 库存 | 多少规格、总共多少件 |
| 库存货值 | 按进价算的库存总价值（自有闲置不计） |
| 本月净利润 | 这个月赚了多少 |

往下还有：

- **分析概览**：切换「本周/上周/本月/上月/自定义」，看卖出件数、成交额、总利润趋势图
- **闲鱼账号概览**：各号的销售情况
- **最近销售**：最新 5 笔成交
- **低库存预警**：快卖完的款

### 8.2 账号分析

想知道 **哪个闲鱼号卖得好**：

1. 进入 **「账号分析」**
2. 选择时间段（如「本月」）
3. 查看：
   - **表现最好**：利润最高的号
   - **需要关注**：出单少或利润偏低的号
   - **柱状图**：各号成交额和利润对比
   - **表格**：详细数据 + 评价标签 + 改进建议

### 8.3 建议的复盘节奏

| 频率 | 做什么 |
|------|--------|
| 每天 | 卖出后及时记账 |
| 每周 | 看数据概览「本周」，关注低库存 |
| 每月 | 看账号分析「本月」，决定是否给某号加流量 |

## 九、典型场景速查

### 场景 1：闲鱼上新一件自有闲置

```
闲鱼导入 → 粘贴分享 → 来源选「自有闲置」→ 选账号 → 保存
```

### 场景 2：批发市场进了一批货

```
商品管理 → 新增商品 → 来源选「进货」→ 填进价和库存 → 保存
（多个规格就加多个 SKU）
```

### 场景 3：闲鱼卖出一单

```
销售记录 → 录入销售 → 选商品 → 选闲鱼号 → 填成交额 → 保存
```

### 场景 4：买家退款了

```
销售记录 → 找到该笔 → 编辑 → 订单状态改「有退款」→ 保存
```

### 场景 5：月底看哪个号最赚钱

```
账号分析 → 选「本月」→ 看「表现最好」和表格排名
```

### 场景 6：某款快卖完了

```
数据概览 → 低库存预警
或
库存管理 → 筛选「低库存」
```

### 场景 7：给闲鱼号买了推广

```
闲鱼账号 → 该号的「流量投入」→ 新增投入 → 填金额
```

## 十、利润怎么算的

不用自己算，系统会自动计算。了解规则有助于核对：

### 自有闲置（自己以前买的）

```
利润 = 成交额 − 平台服务费 − 运费 − 其他费用
（进价按 ¥0 计算）
```

### 进货款

```
利润 = 成交额 − 进价 × 数量 − 平台服务费 − 运费 − 其他费用
```

### 有退款

```
利润 = ¥0
```

### 闲鱼平台费

默认按成交额的 **0.6%** 估算，可在录入时手动修改。

## 十一、常见问题

### Q1：忘记记账了怎么办？

随时补录。进入「销售记录」→「录入销售」，把下单日期改成实际成交日期即可。

### Q2：选商品时找不到？

可能原因：
- 商品状态不是「在售」
- 库存为 0
- 还没录入系统

去「商品管理」检查。

### Q3：录入销售时选不了闲鱼号？

请先去「闲鱼账号」添加至少一个启用的账号。

### Q4：自有闲置和进货有什么区别？

| | 自有闲置 | 进货 |
|---|---------|------|
| 含义 | 自己以前买的 | 为转卖而采购 |
| 进价 | ¥0 | 需填写 |
| 利润 | 成交额减费用 | 还要减进价 |

### Q5：删除销售后库存没回来？

这是有意设计的，防止误删导致库存虚增。如需恢复，去「库存管理」手动入库。

### Q6：闲鱼导入解析失败？

1. 确认分享文案完整（含链接）
2. 点「重新解析」再试
3. 仍失败可手动去「商品管理」新增，把链接和名称自己填上

### Q7：数据概览的数字不对？

检查：
1. 是否所有成交都已录入「销售记录」
2. 闲鱼成交是否选了正确的「成交账号」
3. 退款订单是否标记为「有退款」

### Q8：能同时在手机上用吗？

可以打开浏览器访问，但表格较多，**建议在电脑上用**。

### Q9：聊天和发货要在系统里做吗？

**不用。** 聊天、发货仍在闲鱼 App 完成。这个系统只负责**管货和算账**。

## 十三、H5 随手记（手机同步）

> 详细方案见 [移动端方案.md](./移动端方案.md)

姐姐在闲鱼操作后，用手机打开 H5 页面即可同步，无需开电脑。

| 闲鱼操作 | H5 对应功能 |
|----------|-------------|
| 发布/重新上架 | **上新** → 拍照 + 填信息 |
| 卖出 | **售出下架** → 选商品 + 填成交价 |
| 暂时下架 | **主动下架** → 勾选商品确认 |

启动方式：

```bash
cd mobile && npm run dev   # http://localhost:5174
```

建议部署后把链接发到微信，添加到主屏幕，和闲鱼 App 放一起。

## 十四、快捷操作清单

打印或保存此清单，贴在电脑旁：

```
□ 卖出一件 → 销售记录 → 录入销售
□ 闲鱼上新 → 闲鱼导入 → 粘贴分享 → 保存
□ 新进货   → 商品管理 → 新增 → 来源选「进货」

□ 快卖完   → 数据概览 → 低库存预警
```
